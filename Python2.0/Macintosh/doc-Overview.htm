Download Python for Macintosh

MacPython is already up to Python version 2.0!

Python 2.0 for the Macintosh is available here, thanks to Jack Jansen. In addition to the following, the original developers copies, which may include newer alpha and beta versions, are available from Jack Jansen's ftp archive. Jack also maintains a MacPython page, with information on Python on mac, in general, and his distribution in particular. This page also contains information on accessing the MacPython distribution via anonymous CVS.

For the current, full release, the installer is available in two forms (from Jack's ftp site):

  • MacBinary format (about 6.5 Mb)

  • BinHex format (about 8.5 Mb). Use this installer if your browser mistreats MacBinary files (as many appear to do).

This is a single installer which gives you the option to install classic 68k, cfm68k, and/or PPC. The installer also optionally installs NumPy, PIL, img, Tkinter, and an IDE, and interactive development environment.

Source (including CW Pro1 project files), documentation, older versions, and other odds and ends are available from the same place: ftp://ftp.cwi.nl/pub/jack/python/mac/.
