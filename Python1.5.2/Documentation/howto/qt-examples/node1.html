1. Introduction

Qt is a portable GUI toolkit, described in greater detail at http://www.troll.no/products/qt.html. It's probably best known for its use in KDE (http://www.kde.org), an effort to produce a user-friendly Unix desktop. A tutorial is available at http://www.troll.no/qt/tutorial.html.

A Python/Qt interface is available at ftp://ftp.kde.org/kde/devel/qt-bindings/. The Python/Qt interface is currently without a maintainer, so feel free to pick it up and resume development on it.

This document contains the first four examples from the tutorial, translated from C++ into Python.
