2. xml.parsers.xmllib

This is a version of the xmllib module from Python 1.5, modified to use the sgmlop C extension when it's available. This produces a significant speedup, amounting to about a factor of 5. The interface is unchanged from the original xmllib module; consult the Python Library Reference documentation for that module.
