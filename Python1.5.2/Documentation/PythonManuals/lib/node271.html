18.8 Oddities

These modules are probably also obsolete, or just not very useful.

dump.py
-- Print python code that reconstructs a variable

find.py
-- find files matching pattern in directory tree

fpformat.py
-- General floating point formatting functions -- interesting demonstration of how to do this without using the C library

grep.py
-- grep

mutex.py
-- Mutual exclusion -- for use with module sched

packmail.py
-- create a self-unpacking Unix shell archive

poly.py
-- Polynomials

sched.py
-- event scheduler class

shutil.py
-- utility functions usable in a shell-like program

util.py
-- useful functions that don't fit elsewhere

zmod.py
-- Compute properties of mathematical "fields"

tzparse.py
-- Parse a timezone specification (unfinished)
