This directory contains the standard Python1.5.2
documentation set.  Python's standard manuals are
available in HTML form in the PythonManuals directory;
point your web browser to the index file there.  Note
that on Windows, the manuals are also accessible via 
the Programs/Python entry in your "Start" button.

The file html-1.5.2p2.tgz is an archive containing
the latest releases of the documents available in 
unpacked form in other directories here.
